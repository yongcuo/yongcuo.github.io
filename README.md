# 个人学术主页

这是其美拥措的个人学术主页，展示个人信息、教育背景、校园经历以及医学影像知识图谱。

## 功能特点

- 响应式设计，适配各种设备
- 导航条，方便浏览不同部分
- JavaScript动态效果（滚动动画、交互式知识图谱等）
- 基于磁共振成像的医学影像知识图谱

## 文件结构

- `index.html` - 主页面文件
- `style.css` - 样式文件
- `script.js` - 交互脚本
- `knowledge-graph.js` - 知识图谱脚本

## 部署到GitHub Pages

1. 在GitHub上创建一个名为`[你的用户名].github.io`的仓库
2. 将所有文件上传到该仓库
3. 在仓库设置中启用GitHub Pages功能
4. 访问 `https://[你的用户名].github.io` 查看网站

## 使用的技术

- HTML5
- CSS3
- JavaScript
- D3.js (用于知识图谱)
- Font Awesome (用于图标)
